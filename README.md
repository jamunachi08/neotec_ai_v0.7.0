# Neotec AI v0.7.0

Install-safe cross-module AI automation platform for Frappe / ERPNext.

## Included foundation
- Guided AI Hub workspace
- Universal document AI button
- Feature registry
- Automation recipes
- Data source registry
- Ingestion job registry
- Knowledge asset registry
- Voice command log
- Prompt templates, conversations, action logs
- Prompt-to-report blueprint API
- Safe post-install bootstrap

## Install
```bash
bench get-app <repo-url>
bench --site yoursite install-app neotec_ai
bench build --app neotec_ai
bench clear-cache
```
